Создай HTML-разметку и CSS-стили по описанию макета.

Это страница «История» веб-приложения MoodSync — дневник настроения с анализом текста/NLP.

Нужно сверстать страницу на русском языке, современный минималистичный UI, светлая тема, планшетная/desktop-версия.

Общая структура:
1. Слева фиксированный сайдбар шириной примерно 270px.
2. Справа основная рабочая область на всю оставшуюся ширину.
3. Вверху основной области — горизонтальный header с заголовком страницы и иконками.
4. В центре страницы — список записей дневника настроения, сгруппированный по датам.

Подробное описание макета:

Левый сайдбар:
- Белый фон.
- Справа тонкая вертикальная граница.
- Вверху блок профиля:
  - круглый аватар пользователя;
  - название приложения: MoodSync;
  - под ним небольшой текст: «Ежедневный дневник настроения».
- Ниже вертикальное меню с иконками и пунктами:
  - «Главная» — иконка домика;
  - «История» — активный пункт, светло-серый фон, иконка календаря;
  - «Тренды» — иконка графика;
  - «Аналитика» — иконка диаграммы;
  - «AI-инсайты» — иконка искр.
- Внизу сайдбара, отделённое линией:
  - «Настройки» с иконкой шестерёнки;
  - «Помощь» с иконкой вопроса.

Верхний header:
- Высота около 70px.
- Белый фон.
- Нижняя тонкая граница.
- Слева крупный заголовок страницы: «История».
- Справа иконка фильтра.
- Рядом иконка поиска.

Основная область:
- Фон очень светлый, почти белый с лёгким розоватым оттенком: #faf7f8 или похожий.
- Контент расположен с отступами 36–40px.
- Список записей расположен вертикально.
- Записи сгруппированы по датам.

Группы записей:
- Над каждой группой есть заголовок даты маленькими буквами uppercase:
  - «СЕГОДНЯ, 24 ОКТЯБРЯ»
  - «ВЧЕРА, 23 ОКТЯБРЯ»
- Текст даты должен быть серым, маленьким, с letter-spacing.

Карточка записи:
- Белый фон.
- Border-radius примерно 12px.
- Тонкая серая рамка.
- Лёгкая тень или без сильной тени.
- Высота примерно 150–180px.
- Ширина почти на всю рабочую область.
- Внутри карточка делится на две зоны:
  1. Слева узкая колонка с временем, цветной точкой настроения и бейджем эмоции.
  2. Справа основная часть с заголовком, текстом и тегами.

Левая колонка карточки:
- Время записи сверху:
  - «14:30»
  - «09:00»
  - «21:15»
- Рядом или слева от времени маленькая цветная точка:
  - спокойствие — светло-голубая;
  - радость — светло-жёлтая;
  - усталость — светло-серая.
- Внизу левой колонки pill-бейдж настроения:
  - «Спокойствие»
  - «Радость»
  - «Усталость»
- Между левой и правой частью — вертикальная тонкая линия.

Правая часть карточки:
- Заголовок записи жирным:
  - «Рабочий фокус»
  - «Утренняя прогулка»
  - «Вечернее чтение»
- Ниже короткий текст записи серым цветом.
- Внизу теги в виде маленьких светлых pill-кнопок.

Требования к HTML/CSS:
- Использовать обычные файлы:
  - history.html
  - css/style.css
- Без React, без Bootstrap.
- Можно использовать Font Awesome или Lucide CDN для иконок.
- Сделать адаптивность:
  - на ширине меньше 900px сайдбар должен стать верхним меню или компактным блоком;
  - карточки должны занимать 90–100% ширины;
  - внутренняя сетка карточки может стать вертикальной.
- Использовать аккуратные CSS-переменные:
  - --bg
  - --sidebar-bg
  - --text
  - --muted
  - --border
  - --accent
- Весь текст должен быть на русском языке.
- Нужно, чтобы результат визуально был максимально похож на макет.
